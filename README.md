# 📝 Список дел

Простой список дел на **React + TypeScript**. Позволяет планировать и отслеживать повседневные задачи.

## 🚀 Функционал

- Добавление новых пунктов
- Отметка задач как выполненных
- Удаление задач
- Отображение состояния списка (пустой / заполненный)

## 🎯 Цель проекта

Практика работы с React и TypeScript:
- типизация состояния, пропсов, обработчиков событий
- работа с компонентной архитектурой

## 🛠️ Технологии

- React
- TypeScript
- CSS Modules

## 🚀 Запуск проекта

### Установка зависимостей
```sh
npm install
```

### Запуск в режиме разработки
```sh
npm run dev
```

### Сборка проекта для рабочей версии
```sh
npm run build
```
