# 📝 Список дел

Приложение для управления задачами на **React + TypeScript** с хранением данных в **Supabase**.

## 🌐 Beб-сайт

Поэкспериментировать со списком дел можно [здесь](https://todos-react-ts-navy.vercel.app/).

## 🚀 Функционал

- Добавление в список новых пунктов 
- Отметка задач как выполненных
- Удаление задач
- Сохранение данных на сервере
- Отображение состояний: загрузка / ошибка / пустой список

## 🎯 Цель проекта

Практика работы с React и TypeScript:
- типизация состояния, пропсов, обработчиков событий
- работа с компонентной архитектурой
- работа с REST API (CRUD-операции)
- интеграция с backend-сервисом (Supabase)

## 🛠️ Технологии

- React
- TypeScript
- CSS Modules
- Supabase
- REST API

## 🗄️ Бэкенд

Список дел хранится в базе данных PostgreSQL, управляемой с помощью [Supabase](https://supabase.com/). Фронтенд взаимодействует с сервером через автоматически сгенерированный REST API.

Структура данных:

```ts
interface Todo {
  id: number;
  title: string;
  completed: boolean;
}
```

## 🚀 Запуск проекта

### Установка зависимостей
```sh
npm install
```

### Запуск в режиме разработки
```sh
npm run dev
```

### Сборка проекта для рабочей версии
```sh
npm run build
```
